# JurnalKu 📓

JurnalKu is a modern, intuitive journaling application designed to help you capture your thoughts, track your moods, and express your creativity. Built with the latest Android technologies, it offers a seamless and beautiful experience for personal reflection.

## ✨ Features

-   **🌈 Mood Tracking:** Log your daily emotions and visualize your mood trends over time.
-   **🎨 Custom Journal Canvas:** Express yourself freely with a flexible canvas for your journal entries.
-   **📝 Paper Styles:** Personalize your writing experience with multiple paper styles, including:
    -   **Grid** for structured notes and sketches.
    -   **Dot** for bullet journaling.
    -   **Lined** for classic writing.
-   **📊 Mood Counter:** Get insights into your emotional well-being with an integrated mood counter.
-   **🔐 Secure & Synced:** (Coming soon/Implemented) Keep your journals safe and accessible across devices using Firebase authentication and storage.

## 🛠 Tech Stack

JurnalKu leverages modern Android development tools and libraries:

-   **[Jetpack Compose](https://developer.android.com/jetpack/compose):** Modern toolkit for building native UI.
-   **[Material 3](https://m3.material.io/):** The latest version of Google's open-source design system.
-   **[Navigation Compose](https://developer.android.com/jetpack/compose/navigation):** For seamless screen transitions.
-   **[Room Database](https://developer.android.com/training/data-storage/room):** Robust local data persistence.
-   **[Firebase](https://firebase.google.com/):** 
    -   **Authentication:** Secure user sign-in.
    -   **Firestore:** Real-time cloud database for syncing.
-   **[Coroutines & Flow](https://kotlinlang.org/docs/coroutines-overview.html):** For efficient asynchronous programming.
-   **[Hilt/Dagger](https://developer.android.com/training/dependency-injection/hilt-android):** (Assumed) For dependency injection.

## 🚀 Getting Started

### Prerequisites

-   Android Studio Koala or newer.
-   JDK 17 or higher.
-   A Firebase project (if setting up backend features).

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/JurnalKu.git
    ```
2.  **Open in Android Studio:**
    Open the project folder and let Gradle sync.
3.  **Firebase Setup (Optional):**
    -   Add your `google-services.json` to the `app/` directory.
    -   Enable Authentication and Firestore in your Firebase console.
4.  **Run the app:**
    Click the **Run** button in Android Studio to deploy to your emulator or physical device.
